Define Once. Generate Infinite Data.

Write a simple JSON schema. Get production-ready data instantly.

schema.json 
[
  {
    "fieldName": "user_id",
    "recordType": "uuid",
    "uniqueRecords": 1000
  },
  {
    "fieldName": "full_name",
    "recordType": "fullName",
    "cleanliness": 1.0
  },
  {
    "fieldName": "role",
    "recordType": "appRole",
    "params": {
      "roles": ["Admin", "Editor"]
    }
  },
  {
    "fieldName": "email",
    "recordType": "email",
    "params": {
      "format": "${full_name}@acme.com"
    }
  }
]
